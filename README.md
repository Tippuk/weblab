# Новостник

Блог на Flask + React. Лабораторные работы 2–6.

## Запуск

```bash
pip install -r requirements.txt
python app.py
```

Бэкенд: http://127.0.0.1:5000

Тестовые аккаунты: `admin@blog.local` / `admin`, `author@blog.local` / `author`

## Фронтенд

```bash
cd frontend
npm install
npm run dev
```

Фронтенд: http://localhost:5173
